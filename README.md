# Season Stats
This project allows users to obtain specific statistics for a team(s) from selected leagues and Seasons(years)

## Features
- Select a League, e.g Premier League
- Select a team e.g, Chelsea
- Select a stat or stats, e.g Shots on Target
- Select a Season, e.g 2015 - 2016
- Get back a nicely formatted CLI output of the stat or stats selected
