# Season Stats
This project allows users to obtain specific a statistic for a team(s) from selected leagues and Seasons(years)

## Features
- Select a League, e.g Premier League
- Select a team e.g, Chelsea
- Select a stat or stats, e.g Shots on Target
- Select a Season, e.g 2015 - 2016
- Get back a nicely formatted CLI output of the stat or stats selected

## Libraries used
- Penaltyblog
- Rich
- Argparse
- Pandas

## Example Usage

```bash
uv run main.py "Chelsea" "Shots On Target" "ENG Premier League" "2025-2026"
```
